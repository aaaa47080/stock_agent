🔗 Pi Network 生態系
PI CryptoMind 是 Pi Network 生態系的一部分。本社群守則與 Pi Network 社群準則 保持一致。嚴重違規行為可能導致您在 Pi Network 生態系中受到限制。

PI CryptoMind 致力於打造安全、友善、高品質的加密貨幣社群。所有用戶都應遵守以下守則。

1. 核心原則

  • 尊重他人：避免人身攻擊、歧視、騷擾行為
  • 誠實透明：不發布虛假資訊或操縱市場數據
  • 保護社群：舉報詐騙、濫用行為，共同維護安全環境
  • 理性討論：歡迎不同觀點，但請以建設性方式表達
  • Pi Network 合規：遵守 Pi Network 服務條款與社群準則

2. 違規類型與嚴重程度

違規類型 說明 示例
spam 垃圾訊息、重複發文、無意義內容 連續發布相同廣告、一句話「頂」
harassment 騷擾、霸凌、人身攻擊 辱罵、跟蹤騷擾、仇恨言論
misinformation 故意散布虛假市場資訊 捏造項目破產消息、假冒官方公告
scam 詐騙、釣魚、傳銷 假投資平台、龐氏騙局、假空投
illegal 違法內容 洗錢、販毒、侵犯智財權
other 其他違反守則行為 未分類的不當行為

違規等級與積分

  • ● 輕微 (Mild)：1 分 - 初次無意違規、輕微不當用語
  • ● 中度 (Medium)：3 分 - 重複垃圾訊息、輕度騷擾
  • ● 嚴重 (Severe)：5 分 - 故意散布假消息、嚴重騷擾
  • ● 極嚴重 (Critical)：30 分 - 詐騙、違法行為

3. 處罰機制

違規積分累計達到以下門檻將觸發自動處罰：

積分門檻 處罰措施 說明
5 分 警告 (Warning) 系統發送警告通知，無其他限制
10 分 暫停 3 天 禁止發文、評論、檢舉
20 分 暫停 7 天 禁止除閱讀外所有互動功能
30 分 暫停 30 天 嚴重違規，長期禁止社群參與
40 分 永久封禁 (Ban) 帳戶永久停用，無法申訴

4. 檢舉系統

4.1 檢舉次數限制

  • Free 用戶：5 次/天
  • Plus 用戶：10 次/天
  • PRO 用戶：15 次/天

4.2 檢舉流程

  1. 點擊違規內容的「檢舉」按鈕
  2. 選擇違規類型（spam, harassment, misinformation, scam, illegal, other）
  3. 評估嚴重程度（mild, medium, severe, critical）
  4. 提供證據說明（截圖、連結、文字描述）
  5. 提交檢舉

4.3 PRO 審核員制度

  • PRO 用戶可擔任社群審核員
  • 每個檢舉案件需 3 位審核員投票
  • 達到 70% 共識（至少 2/3 票同意）即執行處罰
  • 若審核員惡意投票將被取消資格

5. 申訴流程

5.1 誰可以申訴

被處罰（暫停或警告）的用戶可在 7 天內提交申訴。

5.2 如何申訴

  1. 前往「個人設定」→「治理記錄」
  2. 找到爭議處罰記錄，點擊「申訴」
  3. 提供證據說明為何該處罰不當
  4. 等待管理員審核（通常 3 個工作日內回覆）

5.3 申訴結果

  • 申訴成立：撤銷處罰，扣除積分
  • 部分成立：降低處罰等級
  • 申訴駁回：維持原處罰

6. 特殊規則

6.1 論壇發文規範

  • 標題需清晰描述內容，不得使用誤導性標題
  • 禁止發布純圖片無文字說明的帖子
  • 引用他人內容需註明來源
  • 討論幣種時應基於事實，避免無根據的「喊單」

6.2 打賞功能

  • 打賞是自願行為，一旦發送不可撤回
  • 不得以打賞進行賄賂或操縱投票
  • 發現異常打賞模式（刷票、洗錢）將凍結帳戶

6.3 詐騙檢舉特別提醒

  • 檢舉詐騙時請提供具體證據（網站截圖、對話記錄）
  • 被檢舉者有權查看檢舉理由（匿名處理檢舉人身份）
  • 誣告詐騙將受到比一般惡意檢舉更嚴重的處罰

7. 管理員權限

管理員保留以下權力：

  • 直接處罰明顯違規內容，無需 PRO 審核流程
  • 調整違規積分（若發現系統誤判）
  • 永久封禁涉嫌犯罪的帳戶並報警
  • 修改社群守則（重大變更將公告）
  • 向 Pi Network 報告嚴重違規用戶

8. 免責聲明

  • 平台不對用戶發布的內容準確性負責
  • 檢舉系統依賴社群自治，可能存在誤判
  • 被檢舉不代表有罪，需經審核流程
  • 平台不對因治理系統產生的爭議承擔法律責任

🔗 Pi Network Ecosystem
PI CryptoMind is part of the Pi Network ecosystem. These Community Guidelines align with Pi Network Community Guidelines. Serious violations may result in restrictions across the Pi Network ecosystem.

PI CryptoMind is committed to building a safe, friendly, and high-quality cryptocurrency community. All users must comply with these guidelines.

1. Core Principles

  • Respect Others: Avoid personal attacks, discrimination, harassment
  • Honesty & Transparency: No false information or market manipulation
  • Protect Community: Report scams and abuse to maintain a safe environment
  • Rational Discussion: Welcome diverse views, but express them constructively
  • Pi Network Compliance: Follow Pi Network Terms of Service and Community Guidelines

2. Violation Types & Severity

Violation Type Description Examples
spam Spam messages, repetitive posts, meaningless content Repeated ads, one-word "bump" posts
harassment Harassment, bullying, personal attacks Insults, stalking, hate speech
misinformation Deliberately spreading false market info Fabricated bankruptcy news, fake announcements
scam Scams, phishing, pyramid schemes Fake investment platforms, Ponzi schemes, fake airdrops
illegal Illegal content Money laundering, drugs, IP infringement
other Other guideline violations Uncategorized misconduct

Severity Levels & Points

  • ● Mild: 1 pt - First-time unintentional violations, minor inappropriate language
  • ● Medium: 3 pts - Repeated spam, mild harassment
  • ● Severe: 5 pts - Deliberate misinformation, serious harassment
  • ● Critical: 30 pts - Scams, illegal activities

3. Penalty System

Accumulated violation points trigger automatic penalties at these thresholds:

Point Threshold Penalty Description
5 pts Warning System sends warning notification, no other restrictions
10 pts 3-day Suspension Cannot post, comment, or report
20 pts 7-day Suspension All interactions blocked except reading
30 pts 30-day Suspension Serious violation, long-term community ban
40 pts Permanent Ban Account permanently disabled, no appeals

4. Report System

4.1 Report Limits

  • Free Users: 5 reports/day
  • Plus Users: 10 reports/day
  • PRO Users: 15 reports/day

4.2 Report Process

  1. Click "Report" button on violating content
  2. Select violation type (spam, harassment, misinformation, scam, illegal, other)
  3. Assess severity (mild, medium, severe, critical)
  4. Provide evidence (screenshots, links, text description)
  5. Submit report

4.3 PRO Moderator System

  • PRO users can serve as community moderators
  • Each report requires 3 moderator votes
  • 70% consensus (at least 2/3 agree) to execute penalty
  • Malicious voting revokes moderator status

5. Appeal Process

5.1 Who Can Appeal

Users with warnings or suspensions can appeal within 7 days.

5.2 How to Appeal

  1. Go to "Settings" → "Governance Records"
  2. Find disputed penalty, click "Appeal"
  3. Provide evidence why the penalty is unjust
  4. Wait for admin review (usually within 3 business days)

5.3 Appeal Outcomes

  • Appeal Granted: Penalty revoked, points removed
  • Partially Granted: Penalty downgraded
  • Appeal Denied: Original penalty stands

6. Special Rules

6.1 Forum Posting Guidelines

  • Titles must clearly describe content, no clickbait
  • No image-only posts without text explanation
  • Cite sources when quoting others
  • Coin discussions should be fact-based, avoid baseless "shilling"

6.2 Tipping Feature

  • Tipping is voluntary, irreversible once sent
  • No bribery or vote manipulation via tips
  • Abnormal tipping patterns (vote manipulation, money laundering) freeze account

6.3 Scam Reporting Reminder

  • Provide concrete evidence (website screenshots, chat logs)
  • Reported users can view report reasons (reporter identity anonymized)
  • False scam reports receive harsher penalties than general malicious reports

7. Administrator Powers

Administrators reserve rights to:

  • Directly penalize obvious violations without PRO review
  • Adjust violation points (if system misjudgment found)
  • Permanently ban accounts suspected of crimes and report to authorities
  • Modify community guidelines (major changes will be announced)
  • Report serious violators to Pi Network

8. Disclaimers

  • Platform not responsible for accuracy of user-generated content
  • Report system relies on community governance, may have misjudgments
  • Being reported doesn't imply guilt, requires review process
  • Platform not liable for disputes arising from governance system
