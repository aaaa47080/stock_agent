# 🚀 快速開始指南

## 三種使用方式

### 方式 1: Web 聊天界面 (推薦) 🌐

最友好的對話式界面，支持自然語言查詢

```bash
# 啟動 Web 界面
python run_chat.py
```

然後在瀏覽器中打開 `http://localhost:7860`

**支持的查詢範例:**
- "PI 可以投資嗎?"
- "PIUSDT 值得買入嗎?"
- "XRP, PI, ETH 哪些可以投資?"
- "比特幣現在適合進場嗎?"

---

### 方式 2: 命令行演示 💻

在終端中直接對話，無需瀏覽器

```bash
# 啟動命令行版本
python demo_chat.py
```

然後輸入你的問題，例如: `PI 可以投資嗎?`

---

### 方式 3: 傳統命令行 🔧

直接指定交易對進行分析

```bash
# 分析單個交易對
python main.py --symbol BTCUSDT --exchange binance

# 分析 PI
python main.py --symbol PIUSDT

# 分析 XRP
python main.py --symbol XRPUSDT
```

## 📊 智能查詢範例

### 單幣種分析
```
你: PI 可以投資嗎?
→ 系統識別 PI，分析 PIUSDT 現貨和合約市場
```

### 多幣種比較
```
你: XRP, PI, ETH 哪個更值得投資?
→ 系統識別 3 個幣種，分別分析並給出比較建議
```

### 自然語言
```
你: 比特幣最近表現如何?
→ 系統自動轉換為 BTC，進行完整分析
```

## 🔍 系統會分析什麼?

每次查詢都會進行以下分析:

1. **技術分析** 📈
   - RSI, MACD, 布林帶等指標
   - 趨勢判斷
   - 支撐/壓力位

2. **情緒分析** 💭
   - 市場情緒評估
   - 交易量分析

3. **基本面分析** 📊
   - 市值排名
   - 項目基本面

4. **新聞分析** 📰
   - 最新相關新聞
   - 新聞情緒影響

5. **多空辯論** ⚖️
   - 多方觀點
   - 空方觀點
   - 綜合評估

6. **風險評估** 🛡️
   - 風險等級
   - 風險因素
   - 建議倉位

7. **最終決策** 💡
   - 買入/賣出/觀望
   - 具體建議
   - 操作策略

## ⚙️ 系統要求

- Python 3.12+
- 虛擬環境已激活
- `.env` 文件已配置:
  ```env
  OPENAI_API_KEY=你的密鑰
  BINANCE_API_KEY=你的密鑰
  BINANCE_API_SECRET=你的密鑰
  ```

## ⚠️ 注意事項

1. **僅供參考**: 本系統提供的分析僅供參考，不構成投資建議
2. **投資風險**: 加密貨幣投資有風險，請謹慎決策
3. **幣種限制**: 僅支持在 Binance 上架的幣種
4. **API 限制**: 請確保 API 密鑰有足夠的調用額度
