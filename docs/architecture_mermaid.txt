【核心檔案說明】

1. core/agents.py     - 包含所有 Agent 實作（分析師團隊、研究員團隊、交易員、風險管理、基金經理及 CryptoAgent）
2. core/tools.py      - 定義 4 個 @tool 工具函數
3. core/graph.py      - LangGraph 工作流定義 (7 個節點)
4. core/agents.py     - 分析師類別 (技術/情緒/基本面/新聞)
5. interfaces/chat_interface.py - Gradio 聊天介面
6. api_server.py      - FastAPI 伺服器

================================================================================

【運作原理簡述】

1. 用戶輸入問題
2. CryptoAgent 將問題傳給 GPT-4o
3. GPT-4o 判斷是否需要調用工具：
   - 不需要 → 直接回答
   - 需要 → 選擇適當的工具
4. 工具執行後返回結果
5. GPT-4o 整理結果生成最終回應
6. 回應串流輸出給用戶
