# 社群治理系統 - 用戶指南

## 概述

社群治理系統是一個基於去中心化理念的社區監督平台，讓 PRO 會員共同維護社區健康。透過內容檢舉、PRO 審核投票、違規點數追蹤和透明的審計日誌，我們共同打造一個安全、友善的交流環境。

## 功能特色

### 🔍 內容檢舉
任何用戶都可以檢舉違規內容（文章、評論），由 PRO 會員組成的審核團隊進行投票決議。

### ⚖️ PRO 審核
PRO 會員可以對檢舉進行投票，累積聲望分數，影響投票權重。

### ⚠️ 違規點數系統
- **輕微違規**: 1 點
- **中等違規**: 3 點
- **嚴重違規**: 5 點
- **極嚴重違規**: 30 點

### 📊 處罰機制
| 累積點數 | 處罰 |
|---------|------|
| 1-4 點 | 無處罰 |
| 5-9 點 | ⚠️ 警告 |
| 10-19 點 | 🚫 暫停 3 天 |
| 20-29 點 | 🚫 暫停 7 天 |
| 30-39 點 | 🚫 暫停 30 天 |
| 40+ 點 | ⛔ 永久停權 |

### 🔄 點數遞減
- 30 天內無新違規且上次遞減超過 30 天，點數將自動遞減 1 點
- 每日自動執行，幫助用戶恢復良好信用

## 使用指南

### 📝 提交檢舉

#### 如何提交檢舉

1. **在文章頁面**
   - 點擊文章下方的「🏳 Report」按鈕
   - 選擇檢舉類型
   - 填寫詳細說明（選填）
   - 提交檢舉

2. **在治理儀表板**
   - 前往 `/web/governance/index.html`
   - 點擊「提交檢舉」標籤
   - 填寫內容類型、ID、檢舉類型
   - 提交檢舉

#### 檢舉類型說明

| 類型 | 說明 | 範例 |
|------|------|------|
| 垃圾信息 | 重複發布、無意義內容 | 重複貼文、刷屏 |
| 騷擾 | 攻擊性語言、人身攻擊 | 辱罵、歧視性言論 |
| 虛假信息 | 散布不實資訊 | 謠言、偽造數據 |
| 詐騙 | 詐騙行為、釣魚 | 假投資、釣魚連結 |
| 違法內容 | 違反法律內容 | 色情、暴力、非法交易 |
| 其他 | 其他違規行為 | 未涵蓋的違規類型 |

#### 檢舉限制

- **免費用戶**: 每日 5 次檢舉
- **PRO 會員**: 每日 10 次檢舉
- 同一用戶對同一內容只能檢舉一次
- 無法檢舉自己的內容

### ⚖️ PRO 審核員

#### 成為 PRO 審核員

升級為 PRO 會員即可獲得審核資格：
- 對待審核檢舉進行投票
- 累積審核聲望
- 影響社區治理方向

#### 投票機制

1. **進入審核隊列**
   - 前往治理儀表板
   - 點擊「審核隊列」標籤（PRO 專用）

2. **查看檢舉詳情**
   - 閱讀檢舉內容和描述
   - 判斷是否構成違規

3. **投表決**
   - 👍 **認為違規**: 同意檢舉
   - 👎 **認為不違規**: 拒絕檢舉

4. **共識機制**
   - 最少 3 票才會結算
   - ≥70% 投票認為違規 → 通過
   - ≤30% 投票認為違規 → 拒絕
   - 否則繼續等待更多投票

#### 聲望系統

| 聲望分數 | 投票權重 |
|---------|---------|
| 0-49 | 1.0x (基礎權重) |
| 50-79 | 1.5x (經驗審核員) |
| 80+ | 2.0x (頂級審核員) |

**聲望計算**:
- 正確投票（與最終結果一致）: +1 分
- 錯誤投票（與最終結果相反）: -1 分
- 準確率 > 90% 可獲得額外權重加成

## 常見問題 (FAQ)

### Q1: 如何檢舉評論？
A: 在評論旁點擊檢舉按鈕（如可用），或前往治理儀表板手動輸入評論 ID。

### Q2: 檢舉後多久會有結果？
A: 視 PRO 審核員投票速度而定，通常在數小時到數天內完成審核。

### Q3: 如果不同意審核結果怎麼辦？
A: 您可以聯繫管理員進行申訴，提供更多證據說明您的情況。

### Q4: 點數會自動恢復嗎？
A: 會。30 天內無新違規，點數將每日自動遞減 1 點。

### Q5: PRO 會員有特權嗎？
A: PRO 會員的審核投票權重更高，且每日檢舉配額更多（10 次 vs 5 次）。

### Q6: 惡意檢舉會受罰嗎？
A: 會。系統會追蹤檢舉準確率，惡意檢舉可能導致您的帳號受罰。

### Q7: 如何成為 PRO 會員？
A: 前往 `/forum/premium.html` 了解升級方案。

### Q8: 被停權後如何恢復？
A: 等待暫停期結束後，帳號將自動恢復。永久停權無法恢復。

## 社區規則

### 禁止行為

1. **發布垃圾信息**
   - 重複發布相同或類似內容
   - 無意義的刷屏行為
   - 廣告或推銷（未經授權）

2. **騷擾行為**
   - 人身攻擊、辱罵
   - 種族、性別、宗教歧視
   - 跟蹤、騷擾其他用戶

3. **散布虛假信息**
   - 謠言、未經證實的指控
   - 偽造或篡改數據
   - 誤導性的投資建議

4. **詐騙行為**
   - 假投資、釣魚連結
   - 盜用他人身份
   - 詐欺性索取資金或信息

5. **違法內容**
   - 色情、暴力內容
   - 非法交易或服務
   - 侵犯版權或商標

### 鼓勵行為

- ✅ 尊重他人觀點
- ✅ 建設性討論
- ✅ 分享有價值的分析
- ✅ 幫助新用戶融入社區
