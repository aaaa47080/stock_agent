# Taiwan Stock Agent

## When to Use
當用戶查詢涉及：
- 台灣公司的股票
- 台股股票代號（數字格式）
- 台灣證券交易所相關資訊
- 台灣上市/上櫃公司財務數據

## Capabilities
- 即時股價查詢
- 技術指標分析 (RSI, MACD, KD, 均線)
- 基本面數據 (本益比, EPS, ROE)
- 三大法人籌碼分析
- 台股新聞
- 月營收、股利資訊

## Market Indicators
用戶提到台灣公司名稱、台股代號、或明確表示查詢台股市場時，應路由至此 agent。
