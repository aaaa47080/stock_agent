# US Stock Agent

## When to Use
當用戶查詢涉及：
- 美國公司的股票
- 美股股票代號（如 AAPL、TSLA、NVDA）
- 美國證券交易所相關資訊
- 美股上市公司財務數據
- 美股 ADR（美國存託憑證）

## Capabilities
- 即時股價查詢（約 15 分鐘延遲）
- 技術指標分析
- 基本面數據
- 財報數據與行事曆
- 機構持倉
- 內部人交易
- 美股新聞

## Market Indicators
用戶提到美國公司名稱、美股代號、或明確表示查詢美股市場時，應路由至此 agent。
