# Commodity Agent

## When to Use
當用戶查詢涉及：
- 貴金屬（黃金、白銀等）
- 能源（原油、天然氣等）
- 工業金屬（銅、鋁等）
- 其他大宗商品

## Capabilities
- 即時商品價格
- 期貨價格
- 金銀比分析
- 原油市場分析

## Market Indicators
用戶提到「黃金」、「白銀」、「原油」、「石油」或明確表示查詢大宗商品時，應路由至此 agent。
